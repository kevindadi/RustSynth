这是一个简单的 Rust "Hello, World!" 函数：

```rust
fn main() {
    println!("Hello, World!");
}
```

**解释：**

1. `fn main()` - 定义了一个名为 `main` 的函数，这是 Rust 程序的入口点
2. `println!("Hello, World!");` - 使用 `println!` 宏打印文本到控制台
   - `!` 表示这是一个宏而不是普通函数
   - 字符串用双引号包裹
   - 语句以分号结尾

**运行方法：**

1. 创建一个名为 `main.rs` 的文件
2. 将上面的代码复制到文件中
3. 在终端中运行：
   ```bash
   rustc main.rs  # 编译
   ./main         # 运行（Windows 上是 main.exe）
   ```

或者使用 Cargo（Rust 的包管理器）：
```bash
cargo new hello_world
cd hello_world
cargo run
```

**更完整的示例（带注释）：**

```rust
// 这是主函数，程序从这里开始执行
fn main() {
    // 使用 println! 宏打印文本到标准输出
    println!("Hello, World!");
    
    // 也可以打印变量
    let name = "Rust";
    println!("Hello from {}!", name);
}
```

**输出：**
```
Hello, World!
Hello from Rust!
```

这个简单的示例展示了 Rust 的基本语法结构，包括函数定义、宏调用和变量声明。
