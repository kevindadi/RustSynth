## 📞 问题和讨论

### 问题 1: 如何处理多个生命周期参数？

```rust
fn complicated<'a, 'b>(x: &'a T, y: &'b U) -> &'a T {
    x  // 返回值绑定到 'a，不绑定到 'b
}
```

**建议**: 从函数签名提取所有生命周期，只处理与返回值相关的绑定。

### 问题 2: Petri 网如何表示函数调用栈？

**方案 A**: 隐式栈（当前实现）

- 不显式建模调用栈
- 假设每个 API 调用都是独立的

**方案 B**: 显式栈

- 添加 enter_fn/exit_fn 变迁
- 用额外的 place 表示当前上下文

**方案 C**: 混合

- 简单函数使用隐式栈
- 复杂函数（如递归）使用显式栈

### 问题 3: 如何测试生命周期系统？
